# 环境配置

所有配置仅供参考，且尽可能使用统一的包管理器安装。Windows 上推荐使用 Chocolatey (opens new window)，macOS 上推荐使用 Homebrew (opens new window)，均不推荐换源。以下给出在 macOS (intel) 上的应用和配置参考，并补充说明在 Windows 上大致如何操作。

# Update formulaes and casks
brew upgrade && brew cu -ay && brew cleanup
# Check problems

# Windows 补充说明

Windows 和 macOS 生态不同，有几个重要的地方需要注意。

  • 终端和 shell：Windows 上建议使用 Windows Terminal (opens new window) 作为终端，Git Bash 作为 shell，定制参考 1 (opens new window)，定制参考 2 (opens new window)。
  • 包管理器：macOS 上使用 Homebrew 作为包管理器，而 Windows 上推荐使用 Chocolaty (opens new window) 作为包管理器。
  • nvm：Windows 上需要使用 nvm-windows (opens new window) 管理 node 版本，而且命令上有一些区别。
  • 其它：上面的应用在 Windows 上不一定有或者有更好的选择，比如 iina 可以换成 PotPlayer (opens new window)，网易 Mumu 可以换成 雷电模拟器 (opens new window)，Tencent Lemon 可以换成 腾讯电脑管家 (opens new window)，万彩办公大师 (opens new window) 可以处理大量的文档等。
